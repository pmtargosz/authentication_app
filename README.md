# Overview

Playground for authentication, cookie, token, nodejs, express, mongoose, api and content server.

## Installation

First, clone this repo and change to the directory:

```bash
git clone git@github.com:pmtargosz/<project>.git
cd <project>
```

### Install

```bash
cd <project>
npm install
```

### Run

```bash
cd <project>
npm start
# http://localhost:3000
```
